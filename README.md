# NetSecureX - SOC Dashboard

## Overview
NetSecureX is a multithreaded TCP port scanner built using Python.
It provides a graphical dashboard for scanning open ports, classifying risk levels, and generating reports.

## Features
- Quick Scan (1–1024 ports)
- Full Scan (1–5000 ports)
- Multithreaded scanning for high speed
- Risk classification (HIGH, MEDIUM, LOW)
- Banner grabbing
- Real-time scan dashboard
- Exportable scan reports

## Technologies Used
- Python
- Socket Programming
- Multithreading
- Tkinter (GUI)

## How to Run
1. Install Python 3.x
2. Open terminal in project folder
3. Run:

   python main.py
